-SELECT                                 (colname,colname,...)

-WHERE                                  (logical condition)
-GROUP BY                               (colname)->(segment data for aggregate level analysis)
-HAVING                                 (logical condition) -> (group filtering criteria)
-ORDER BY                               (colname)->(for sorting)
